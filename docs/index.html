Rock-n-slide is a set of scripts with gulp that generates site like slideshow or presentation. Edit data.json to make your slideshow and roll it.

JSON

Rock-n-slide API

API with default settings

  • speed - changing slide animation speed
  • lazyLoad - lazy load of slide images
  • animateSliding - animate slides
  • help - dependencies run informations, prints in console if dependencies has been loaded
  • classList
    • slidesWrapper - class name of all slides wrapper
    • slide - slide class name
    • menuTrigger - hamburger/menu trigger class name
    • menuTriggerActive - active class name for hamburger/menu
    • menuSlide - class of slide that contains menu
    • menuSlideActive - active class of slide that contains menu
    • menuItem - class of menu item
    • scrollText - scrolling text class name, appears on first slide and hides during scrolling down
